# AZ-204: Developing Solutions for Microsoft Azure - Complete Study Guide (Synthetic)

## Certification Overview
- **Exam Code**: AZ-204
- **Level**: Associate
- **Duration**: 120 minutes
- **Passing Score**: 700/1000
- **Target Role**: Cloud Developers and Engineers

## Core Skill Areas

### 1. Develop Azure Compute Solutions (25-30%)
- Develop Azure App Service Web Apps
- Develop Azure Functions
- Develop solutions using Azure Container Instances and Azure Kubernetes Service
- Implement Azure Container Apps

### 2. Develop for Azure Storage (15-20%)
- Develop solutions using Blob Storage
- Develop solutions using Azure Cosmos DB
- Develop solutions using Azure Table Storage and Azure Queue Storage

### 3. Implement Azure Security (15-20%)
- Implement authentication and authorization
- Implement secure cloud solutions using Azure Key Vault
- Implement Managed Identities

### 4. Monitor, Troubleshoot, and Optimize Azure Solutions (15-20%)
- Integrate caching and content delivery
- Instrument solutions to support monitoring and logging
- Troubleshoot and debug Azure solutions

### 5. Connect to and Consume Azure Services and Third-Party Services (15-20%)
- Implement API Management
- Develop event-based solutions
- Develop message-based solutions
